## Overview

A complete Phase II **randomized controlled trial (RCT)** statistical analysis in R — from Statistical Analysis Plan (SAP) through final efficacy tables and survival analysis. The study evaluates antiviral treatment (ART-X) vs. placebo in 200 HIV-1–infected adults, applying the full biostatistics workflow used in **peer-reviewed clinical research and FDA submissions**.

All code is reproducible using the included simulated dataset (200 patients, 2 arms) structured to reflect real ACTG/INSIGHT trial data conventions.

## Pipeline

```
Study Design & SAP
  ├── Protocol definition (Phase II, 1:1 randomization, double-blind)
  ├── Stratified randomization (CD4 stratum × sex)
  └── Power analysis (pwr package) — 80% power, α = 0.05
          │
          ▼
Data Simulation (200 patients)
  ├── Baseline: age, sex, CD4 count, viral load (log10)
  ├── Primary outcome: treatment response at Week 24 (binary)
  ├── Survival outcome: time to viral suppression (exponential model)
  ├── Protocol deviations → per-protocol exclusions
  └── Introduced missingness (15% CD4, 10% VL) for MICE
          │
          ▼
CONSORT 2010 Flow Diagram
          │
          ▼
Table 1 — Baseline Characteristics (tableone, SMD balance check)
          │
          ▼
Primary Efficacy Analysis (ITT)
  ├── Chi-square test (unadjusted response rates)
  ├── Multivariable logistic regression (adjusted OR, 95% CI)
  └── Forest plot of adjusted odds ratios
          │
          ▼
Per-Protocol Sensitivity Analysis
          │
          ▼
Survival Analysis
  ├── Kaplan-Meier curves (survminer, risk table)
  ├── Log-rank test
  ├── Cox proportional hazards regression (adjusted HR)
  └── Proportional hazards assumption check (Schoenfeld residuals)
          │
          ▼
Missing Data — Multiple Imputation (MICE)
  ├── m = 20 imputations, predictive mean matching
  ├── Pooled estimates via Rubin's rules
  └── Fraction of Missing Information (FMI) reported
          │
          ▼
Subgroup Analyses (Forest Plot)
  └── Sex, age, CD4 stratum, baseline VL
          │
          ▼
Efficacy Summary Table + Clinical Conclusions
```

## Statistical Methods

| Method | Implementation | Purpose |
|--------|---------------|---------|
| Power analysis | `pwr::pwr.2p.test()` | Sample size justification |
| Table 1 | `tableone::CreateTableOne()` with SMD | Baseline balance assessment |
| Chi-square test | `stats::chisq.test()` | Unadjusted primary endpoint |
| Logistic regression | `stats::glm(family=binomial)` | Adjusted ORs — primary analysis |
| Kaplan-Meier | `survival::survfit()` | Time-to-event estimation |
| Log-rank test | `survival::survdiff()` | Survival curve comparison |
| Cox PH regression | `survival::coxph()` | Adjusted HRs — survival |
| PH assumption | `survival::cox.zph()` | Schoenfeld residual test |
| Multiple imputation | `mice::mice(method='pmm')` | Missing data handling |
| Pooled estimates | `mice::pool()` | Rubin's rules combining |
| Subgroup forest plot | `ggplot2` | Heterogeneity visualization |

## Key Results (Simulated Dataset)

| Endpoint | ART-X | Placebo | Test Statistic |
|----------|-------|---------|---------------|
| Response at Week 24 | ~65% | ~35% | χ² p < 0.001 |
| Adjusted OR (treatment) | ~3.5 [2.0, 6.0] | Ref | Logistic regression |
| Median time to suppression | ~14 weeks | ~30 weeks | Log-rank p < 0.001 |
| Adjusted HR (treatment) | ~2.1 [1.5, 2.9] | Ref | Cox PH |

## Clinical Relevance

- **Intent-to-Treat (ITT)** as primary estimand — aligns with ICH E9(R1) guidelines and FDA expectations
- **CONSORT 2010** flow diagram for participant disposition — required for journal submission
- **SAP finalized before unblinding** — demonstrates protocol statistician workflow
- **MICE with Rubin's rules** — standard for handling missing data in regulatory submissions
- **Schoenfeld residual test** for proportional hazards assumption — required in any Cox regression report
- **Kaplan-Meier + log-rank + Cox** — the complete survival analysis triad used in ACTG, INSIGHT, and DAIDS-sponsored HIV trials
